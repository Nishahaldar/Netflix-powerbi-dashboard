# Netflix Power BI Dashboard

## Project Overview
This project is an interactive Power BI dashboard that analyzes Netflix content including movies and TV shows.  
It helps understand trends in ratings, directors, countries, and release years.

## Dataset
The dataset contains information about Netflix titles such as:
- Show ID
- Title
- Director
- Country
- Release Year
- Rating
- Duration
- Genre

## Tools Used
- Power BI
- Excel / CSV Dataset
- Data Visualization

## Key Insights
- Total Movies and TV Shows on Netflix
- Rating distribution of content
- Top directors producing content
- Content distribution by country
- Release year trends
- Movie duration analysis
